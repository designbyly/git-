## Git优势

- 大部分操作在本地完成，不需要联网。
- 完整性保证。
- 尽可能添加数据而不是删除或修改数据。
- 分支操作非常快捷流畅。
- 与Linux命令全面兼容。

## Git结构

- 本地库   （历史版本）
- 暂存区   （临时存储）
- 工作区    （撸代码）
